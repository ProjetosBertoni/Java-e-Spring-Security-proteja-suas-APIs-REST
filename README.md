# 🔐 API REST Segura com Spring Security

Projeto desenvolvido durante o curso **Java e Spring Security: proteja suas APIs REST** da Alura.

## 📚 Sobre o Projeto

Esta aplicação demonstra a implementação de autenticação e autorização em uma API REST utilizando:

- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- JPA / Hibernate
- Banco de dados relacional

O foco principal é proteger endpoints utilizando autenticação stateless baseada em token JWT.

---

## 🚀 Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Maven
- Banco H2 / PostgreSQL (dependendo da configuração)

---

## 🔐 Segurança Implementada

- Autenticação via login
- Geração de token JWT
- Validação de token a cada requisição
- Autorização baseada em Roles
- Configuração de SecurityFilterChain
- API Stateless
- Senhas criptografadas com PasswordEncoder

---

## 🧠 Conceitos Aplicados

- Inversão de Controle (IoC)
- Injeção de Dependência
- Filtros de Segurança
- UserDetails e UserDetailsService
- AuthenticationManager
- OncePerRequestFilter
- Controle de acesso por perfil (ROLE)
